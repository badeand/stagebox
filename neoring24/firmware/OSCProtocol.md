

## single

Set the RGB value for a specific pixel.

### Parameters

| Name | Meaning | Type | Range |
|------|---------|------|-------|
| Index | Index for the neopixel to control | integer | 0 - ( number of pixels -1 )
| R | Red | integer | 0 - 255
| G | Red | integer | 0 - 255
| B | Red | integer | 0 - 255
